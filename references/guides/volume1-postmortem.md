## 核心经验：字数控制

### 问题
Writer Agent容易过度分割段落（每段1-2句），导致字数远超目标。第2章写了4182字，压缩后3011字。

### 解决方案
1. Writer prompt中明确限制："每段不超过3行"
2. 超过3500字时调用Rewriter定向压缩而非重写
3. 保留核心剧情和关键对话，删掉重复对话和冗余心理描写
4. **绝对禁止AI估算字数**——必须用Python正则 `re.findall(r'[\u4e00-\u9fff]', content)`

### 压缩优先级（从高到低）
1. 环境描写冗余
2. 重复段落（两段表达相同信息的对话/旁白，删其一）
3. 过渡句精简
4. 对话精简（删无信息量的动作描述如"他顿了顿"）
5. 心理描写压缩

### 不可压缩的部分
- 核心打脸/反转场景
- 关键悬念钩子（章节结尾）
- 角色标志性动作/台词

## 核心经验：去AI痕迹

### Writer自带约束优于事后润色
Writer/Rewriter prompt中已内置去AI规则，源头写好比事后润色省力得多。De-AI环节主要做质量检查而非大规模改写。

### 高频AI痕迹模式清单
| 模式 | 原始次数 | 修复策略 |
|------|:---:|------|
| "盯着...看了X秒/分钟/很久" | 31 | 多样化表达：目光落在...、久久地看着... |
| "确认...确认...确认"排比 | 1处 | 改为自然内心独白 |
| 去AI痕迹报告元数据泄露 | 2章(ch016,ch028) | 报告独立存为.deai_report_NNN.md |
| 手部攥拳/握拳动作重复 | 多处 | 分散使用不同身体语言 |

### 去AI痕迹核心规则
- 保留剧情不变，只改表达方式
- 消除模板化开头（"XX睁开眼睛"等）
- 消除万能套话（"心中涌起暖流"等）
- 情绪用行动表现，不用形容词直接说
- 对话口语化，加入语气词、省略句
- 加入感官细节（气味、温度、触感）
- 打破句式重复，长短句交替

## 核心经验：现实实体名合规

### 为什么必须替换
番茄2026年签约审核对现实实体名称极其严格：
- 真实企业名可能触发商标/版权侵权
- 真实媒体名属于他人商业标识
- 编辑认为"使用现实品牌降低沉浸感"

### 高风险词（必须替换）
| 原文 | 替换为 | 说明 |
|------|--------|------|
| 楚天都市报 | 江城都市报 | 剧情核心角色，需完全虚构化 |
| 武汉大学 | 江城大学 | 关键场景地点 |
| 抖音 | （删除） | 2016年上线，2005年不存在 |
| 拼多多 | （删除） | 2015年上线，2005年不存在 |
| 武汉/汉阳/汉口/武昌 | 江州/江州区/江州三阳路 | 湖北第二大城市 |
| 广州 | 海城 | 华南市场 |
| 深圳 | 鹏城 | 科技创业地 |
| 成都 | 锦城 | 西南地区 |
| 上海 | 申城 | 华东市场 |
| 香港 | 港岛 | 海外资金通道 |

### 虚构城市命名规范
全书统一使用：江城（主角大本营）、江州（原武汉）、江城大学、江城都市报、江城电视台、江城晚报。
详细映射表见各书目录下的 `entities_mapping.md`。

## 核心经验：时代错误校验

### 2005年背景不能出现的术语/品牌
- 抖音（2016）、拼多多（2015）、微信（2011）
- 等保三级认证（2007年后）、ICP许可证（语境应为"手续"）
- 德勤审计报告 → 会计师事务所审计报告
- 智能手机/APP/APP Store、云计算/AI大模型/区块链
- 英文化表达（如"Knowing的感觉""cover不住"）

## 核心经验：中段爽点密度提升

### 最有效方式：插入中型打脸/反杀场景（第10-20章区间）

**电话反杀模式**：
1. 对手来电炫耀
2. 主角从容揭露对方底牌（空壳公司、解约信息、替代供应链）
3. 当场碾压挂断

**关键要素**：
- 提前铺垫信息源（如"陈默告诉我的"）让反击有据可依
- 三组反击信息压缩在一通电话内快速输出
- 结尾用细节收束（嘴角弯了一下，没出声）

## 核心经验：章节末尾不写字数标注

### 变更原因
- 番茄后台自动识别章节内容，字数标注是流水线内部使用的元数据
- 字数标注不应出现在正文文件中
- 每章完成后用Python正则验证字数即可

### 实施
- 删除30章中所有【本章字数：XXXX字】和【下一章预告：XXX】
- 更新Writer/Rewriter/De-AI prompt中的输出格式要求
- 更新pre-writing-checklist.md

## 剩余问题（不阻塞签约但值得后续改进）

1. **章节结构模板化**：30章结构高度一致（冲突→冷静应对→打脸→震惊→悬念），后续卷次可尝试变化节奏
2. **主角记忆过于精确**：对2005-2015年的记忆精确到百分比和具体数字，缺乏人类记忆的正常模糊性
3. **配角功能化严重**：陈默=技术工具人、苏晚=情报+感情工具人，可增加配角个人故事线
4. **商战线与科幻线过渡**：第26章突然转入量子物理/时空管理局，应在第10-20章间引入更多铺垫
