# 现实实体名合规指南

## 为什么必须替换

番茄小说签约审核对现实实体名称非常严格：
- 真实企业名可能触发商标/版权侵权
- 真实媒体名（报纸、电视台）属于他人商业标识
- 编辑会认为"使用现实品牌降低沉浸感"

## 替换规则

### 必须替换（高风险）
| 原文 | 替换为 | 说明 |
|------|--------|------|
| [真实报纸名] | [虚构报纸名] | 真实报纸，作为剧情核心角色需完全虚构化 |
| [真实高校名] | [虚构大学名] | 真实高校，关键场景地点 |
| 抖音 | （删除） | 2016年上线，{SETTING_YEAR}年不存在 |
| 拼多多 | （删除） | 2015年上线，{SETTING_YEAR}年不存在 |
| 等保三级认证 | 公安局网络安全备案 | {SETTING_YEAR}年不存在此标准 |
| ICP许可证 | 省通信管理局手续 | 同上 |
| 德勤审计报告 | 会计师事务所审计报告 | 德勤是真实品牌 |

### 建议替换（中风险）
| 原文 | 替换为 | 说明 |
|------|--------|------|
| [省级电视台] | [市级虚构电视台] | 省级台争议较小，但虚构更安全 |
| [真实日报名] | [虚构晚报名] | 避免与真实同名报纸冲突 |

### 可以保留（低风险）
| 原文 | 说明 |
|------|------|
| 武汉、北京、广州等城市名 | 地名一般不构成侵权 |
| 工商、公安局、消防等机构名 | 通用概念，非特定实体 |
| QQ（2005年已存在） | 时代正确，但可改为"飞信"更稳妥 |
| [虚构公司名] | 已在 `entities_mapping.md` 中标记为安全 |

## 替换范围

每次替换需要检查：
1. 所有章节正文内容
2. outline.md 中的章节描述
3. characters.md 中的角色背景
4. .sign_assess.md 评估素材（重新生成时会自动包含）

## 虚构城市命名规范

全书统一使用：江城（主角大本营）、江州（原武汉）、江城大学、江城都市报、江城电视台、江城晚报。

详细映射表见各书目录下的 `entities_mapping.md`。

**如果没有 `entities_mapping.md` 文件**，使用以下通用规则：
- 城市名：{CITY_NAME}（对应 {REAL_CITY}）
- 大学：{UNIVERSITY_NAME}
- 报纸：{NEWSPAPER_NAME}
- 电视台：{TV_STATION_NAME}

保持全书地名一致性，不要混用多个虚构城市名。

## 时代错误校验清单

2005年背景不能出现的术语/品牌：
- [ ] 抖音（2016）
- [ ] 拼多多（2015）
- [ ] 等保三级认证（2007年后）
- [ ] ICP许可证（2000年后，但2005年语境应说"手续"而非"许可证"）
- [ ] 德勤/普华永道/安永/毕马威（四大审计在2005年游戏行业极少出现）
- [ ] 微信（2011）
- [ ] 支付宝（2004年上线但2005年几乎无人使用，避免提及）
- [ ] 淘宝（2003年上线但2005年规模小，可提及但要符合时代语境）
- [ ] 百度推广/竞价排名（2001年有，但2005年叫"竞价排名"而非"推广"）
- [ ] 智能手机/APP/APP Store（全部不存在）
- [ ] 云计算/AI大模型/区块链（全部不存在）

